## Dependencies
#### client side:
    axios
    core-js
    vue
    vue-router

#### server side:
    babel/core
    @babel/preset-env
    @types/jest
    babel-jest
    jest"
    nodemon
    supertest


## Getting Started

- Install all dependencies (using the `npm install` command).

## Run Commands:

-- Server
> Run in Dev: `npm run start`

> Run Tests: `npm test`

-- Client

>  Run in Dev: `npm run serve`
